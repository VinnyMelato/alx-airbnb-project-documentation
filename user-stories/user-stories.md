# User Stories

1. As a user, I want to register so that I can use the platform.
2. As a guest, I want to search properties so that I can find accommodation.
3. As a guest, I want to book a property so that I can reserve my stay.
4. As a host, I want to list properties so that guests can book them.
5. As a user, I want to make payments securely so that my booking is confirmed.
